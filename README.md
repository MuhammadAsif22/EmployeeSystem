# Employee Management System (JSON File Storage)

# How to Run the Project

1. **Restore dependencies**
   dotnet restore

2. **Build the project**
   dotnet build

3. **Run the project**
   You can press **F5** in Visual Studio, or run the command below:
   dotnet run

4. **Open Swagger UI in your browser**
   http://localhost:5222/swagger

Swagger allows you to test all available API endpoints or you can test in Postman
